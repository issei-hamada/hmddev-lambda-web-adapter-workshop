# Appendix

Code Server に Amazon Q Developer をインストールし、エージェントコーディングを体験しましょう。

## Amazon Q Developer とは?

Amazon Q Developer は、AWS が提供する生成 AI を活用した開発者向けアシスタントです。コーディング、デバッグ、テスト、ドキュメント作成など、ソフトウェア開発のあらゆる場面で開発者をサポートします。

### 主な特徴

1. **コード生成と補完**
   - 自然言語での指示からコードを自動生成
   - コンテキストを理解した高度なコード補完

2. **コード分析と改善**
   - セキュリティ脆弱性の検出
   - コード品質の向上提案
   - バグの発見と修正案の提示

3. **マルチ言語対応**
   - Python、JavaScript、Java、TypeScript など主要な言語をサポート
   - フレームワークやライブラリの知識も豊富

### Amazon Q Developer CLI について

今回のハンズオンでは、Amazon Q Developer の CLI 版を使用します。

#### CLI の特徴

1. **対話型インターフェース**
   - ターミナル上で AI アシスタントと直接対話
   - ファイルの作成・編集・分析をコマンドラインから実行

2. **プロジェクト全体の理解**
   - ディレクトリ構造を把握してコンテキストに基づいた提案
   - 複数ファイルにまたがる変更も可能

3. **エージェント機能**
   - 複雑なタスクを自律的に実行
   - 必要なファイルの作成・編集を自動で行う

#### 基本的な使い方

- `q` コマンドで対話モードを開始
- 自然言語で指示を入力（日本語対応）
- ファイルの作成・編集・分析が自動実行される

## 1. builders Id の発行

以下ページへアクセスし、builders Id を発行

- <https://profile.aws.amazon.com/>

## 2. インストール

### 2.1 Code Server のターミナルから以下コマンドを入力し、Amazon Q Developer をインストール

```bash

wget https://desktop-release.q.us-east-1.amazonaws.com/latest/amazon-q.deb

sudo apt-get install -f
sudo dpkg -i amazon-q.deb

```

### 2.2 ログインコマンドを実行し、builders Id を使ってログイン

```bash
q login
```

## 3. Amazon Q Developer を起動

```bash
q
```
